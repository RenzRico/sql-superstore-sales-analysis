# Superstore Sales Analysis (SQL Project)

## 📌 Project Overview
This project analyzes a retail Superstore dataset using SQL to extract
business insights related to sales performance, customer behavior,
and product profitability.

The goal is to demonstrate strong SQL fundamentals commonly required
for Data Analyst and Business Analyst roles.

---

## 🗂 Dataset
- Source: Kaggle – Sample Superstore Dataset
- Records: Orders, Customers, Products, Regions
- Time Period: Multiple years of sales data

---

## 🛠 Tools Used
- SQL (PostgreSQL / MySQL compatible)
- GitHub
- Kaggle Dataset

---

## 📊 Key Business Questions Answered
1. What are the total sales and profit by year?
2. Which regions and states generate the highest revenue?
3. Who are the top 10 customers by total sales?
4. Which product categories are the most profitable?
5. What is the month-over-month sales growth?
6. Which products have high sales but low profit?

---

## 🧠 SQL Skills Demonstrated
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- JOINs (INNER, LEFT)
- Subqueries
- Window Functions (ROW_NUMBER, RANK)
- Date functions
- KPI calculations

---

## 📈 Sample Insights
- The Technology category generates the highest profit margin.
- A small group of customers contributes to a large percentage of total sales.
- Certain products have high sales volume but negative profitability.

---

## ✅ How to Use
1. Download the dataset from Kaggle
2. Import CSV files into your SQL database
3. Run the queries from the `/queries` folder
4. Review insights and results
